# 3. INFORMAÇÕES DO PRODUTO

## 3.1 Software

**Nome do Produto:**  
Manchester Triage App

**Linguagem de Programação:**  
JavaScript (React Native), PHP (Laravel - Backend)

**Banco de dados: (caso haja)**  
MySQL (utilizado pelo backend Laravel)

**Resumo da Aplicabilidade do Software:**  
Sistema mobile de triagem médica baseado no Protocolo Manchester, desenvolvido para auxiliar profissionais de saúde (enfermeiros e médicos) na classificação de risco de pacientes em unidades de emergência. O aplicativo permite registro de dados clínicos, sinais vitais, avaliação de segurança, classificação de risco conforme o protocolo Manchester (vermelho, laranja, amarelo, verde, azul), registro de SAE (Sistematização da Assistência de Enfermagem), atendimento médico e gerenciamento de pacientes.

**Qual problema o software pretende resolver:**  
O software resolve o problema de gestão e classificação de risco de pacientes em unidades de emergência, automatizando e padronizando o processo de triagem conforme o Protocolo Manchester. Facilita o registro rápido e preciso de dados clínicos, sinais vitais e informações de segurança do paciente, permitindo que os profissionais de saúde tomem decisões mais rápidas e precisas sobre a prioridade de atendimento. Reduz erros de classificação, melhora o fluxo de atendimento e garante que pacientes críticos sejam atendidos com a urgência adequada.

**Framework: (caso haja)**  
- React Native (Framework mobile)
- Expo (Plataforma de desenvolvimento React Native)
- Laravel (Framework PHP para backend)
- React Navigation (Navegação)
- Axios (Cliente HTTP)

**Categoria:**  
Aplicativo Mobile / Saúde / Sistema de Gestão Hospitalar / Triagem Médica

**Breve Descrição:**  
Aplicativo mobile multiplataforma desenvolvido em React Native com Expo para triagem de pacientes em unidades de emergência. Utiliza o Protocolo Manchester para classificação de risco (5 níveis: vermelho, laranja, amarelo, verde, azul). Possui interface otimizada para uso em dispositivos móveis com suporte a formulários inteligentes, navegação por etapas, notificações de reavaliação, registro de SAE completo e integração com backend Laravel via API REST. Suporta múltiplos perfis de usuário (Administrador, Enfermeiro, Médico) com permissões diferenciadas.
